# Week 6 – React Hands-on Lab 3

## Task

Created a React app named **scorecalculatorapp**. Inside it, I built a functional component called `CalculateScore`, which displays a student’s name, school, total marks, goal, and calculates the average score. I also applied custom styles using an external CSS file.

## What I Learned

- How to create a **function component** in React
- How to pass and display variables in JSX
- How to create and link a **CSS stylesheet** to a component
- How to organize folders (Components, Stylesheets)
